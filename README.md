# 유은이의 모바일 청첩장
모바일 청첩장을 위한 레파지토리

Vue 로 작업했고 

떨어지는 꽃잎은 scss 로 구현했습니다. 

안드로이드의 모바일웹뷰에서 보았을때 모든 기능이 최적화된 상태로 보입니다.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
